Se preferir, você pode carregar a extensão em Modo desenvolvedor no Chrome.

Instalação manual (modo desenvolvedor)

  1. Abra chrome://extensions no Chrome.
  2. Ative Developer mode (canto superior direito).
  3. Clique Load unpacked e selecione a pasta dist/ da extensão (após rodar o script de build).

Como gerar o pacote (.zip)

# na raiz do repositório
npm i -D archiver
node scripts/build-extension.mjs

O build copia os arquivos de extension-src/ → dist/ e gera dist/extension.zip.

Notas

  • Coloque os arquivos da extensão na pasta extension-src/ (manifest.json, src/, icons/).
  • O script de build requer archiver (devDependency) — instale com npm i -D archiver.
  • O arquivo ZIP será publicado como artefato no CI e também disponível no Pages em /dist/extension.zip (se o branch Pages incluir esse diretório).
